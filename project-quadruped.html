Quadruped Robot Dog

Embedded Control & Programming Interface

Arduino Inverse Kinematics Web-Based UI C++ Embedded Systems

Overview

This project involved designing and implementing a full embedded control platform for a quadruped robot (robot dog), including real-time motor control, inverse kinematics, predefined gait sequencing, and a browser-based programming interface. The goal was to make advanced robotics concepts accessible and interactive — enabling users to live-code robot behaviors through an intuitive, Scratch-style UI without needing deep embedded systems knowledge.

The system bridges hardware and software: the robot's physical motion is governed by inverse kinematics computed on-board, while a web interface allows users to build and execute behavior sequences in real time over a local network connection.

Technical Details

Embedded Control

The robot's joints are driven by servos controlled via an Arduino-based microcontroller. The firmware handles servo PWM signals, reads joint state, and executes motion commands sent over serial or network.

Inverse Kinematics

Analytical inverse kinematics are used to translate end-effector (foot) positions into joint angles for each leg independently, enabling smooth, coordinated gaits and stable stance control.

Gait Sequencing

Predefined gait patterns (walk, trot, etc.) are implemented as timed step sequences, with each gait phase coordinating foot lift, swing, and stance across all four legs.

Browser-Based UI

A Scratch-style visual programming interface runs in the browser, allowing users to drag and drop behavior blocks, assemble sequences, and push them to the robot for live execution — no code required.

Key Challenges

  • Coordinating 12 servo joints in real time with minimal latency on constrained hardware
  • Deriving and validating the per-leg inverse kinematics geometry for stable foot placement
  • Designing a communication protocol between the browser UI and the embedded system
  • Making the programming interface intuitive enough for non-engineers while remaining expressive enough for complex behaviors
